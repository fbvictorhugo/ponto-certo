# Ponto Certo ![ic_launcher](app/src/main/res/mipmap-mdpi/ic_launcher.png)

O PontoCerto é um aplicativo Android desenvolvido para auxiliar no controle de jornada de trabalho diária, garantindo que o usuário cumpra exatamente sua carga horária definida (ex: 8 horas diárias), considerando corretamente o intervalo de almoço.

O app calcula automaticamente o horário ideal de saída com base nos registros realizados ao longo do dia, ajudando a evitar erros na marcação de ponto e excesso de horas trabalhadas.
